剪贴板提示器，用来提示你剪贴板东西的小工具
那么，请问怎么才能用到呢？
* 首先呢，我们下载并解压release中的rar
1.jpg
* 接着运行ClipboardReminder.exe

* 运行成功的话会有消息提示，注意只能运行一个哦（其实错了也没提示）_(:з」∠)_
3.png
* 接着我们随便复制一段文字，就会有提醒
4.jpg
* 截图也是哦
5.jpg
* 位置不合适可以直接用鼠标拉
6.png
